# 📝 TextUtils React App

A simple React application built as a **practice project** to strengthen my React.js, JavaScript, HTML, and CSS skills. The app provides useful text manipulation and analysis features such as case conversion, word and character counting, removing extra spaces, copying text, and live preview.

## 🎯 Purpose

This project was created as part of my learning journey to practice:
- React.js fundamentals
- Component-based architecture
- State management using Hooks
- Event handling
- Responsive UI design
- Deploying React applications using GitHub Pages

## ✨ Features

- Convert text to uppercase / lowercase
- Remove extra spaces
- Copy text to clipboard
- Clear text
- Show word and character count and live preview
- Simple light/dark theme toggle (if implemented)
